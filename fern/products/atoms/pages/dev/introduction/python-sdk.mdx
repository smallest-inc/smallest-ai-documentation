The Atoms SDK is the official Python package for building voice agents on the Smallest AI platform.

```bash
pip install smallestai
```

**Requirements:** Python 3.9+

---

## Core Components

| Component | Description | Learn More |
|-----------|-------------|------------|
| **Nodes** | Processing units that handle events | [Nodes →](/atoms/developer-guide/get-started/core-concepts/nodes) |
| **Events** | Messages flowing through the system | [Events →](/atoms/developer-guide/get-started/core-concepts/events) |
| **Graphs** | Connect nodes into pipelines | [Graphs →](/atoms/developer-guide/get-started/core-concepts/graphs) |
| **Sessions** | Manage conversation state and lifecycle | [Sessions →](/atoms/developer-guide/get-started/core-concepts/sessions) |

---

## Key Modules

| Import | Purpose |
|--------|---------|
| `smallestai.atoms.agent.nodes` | `Node`, `OutputAgentNode`, `BackgroundAgentNode` base classes |
| `smallestai.atoms.agent.events` | Event types (`SDKEvent`, etc.) |
| `smallestai.atoms.agent.tools` | `@function_tool` decorator, registry |
| `smallestai.atoms.agent.clients` | LLM clients (OpenAI-compatible) |
| `smallestai.atoms.agent.server` | `AtomsApp` for hosting |
| `smallestai.atoms.agent.session` | `AgentSession` management |
