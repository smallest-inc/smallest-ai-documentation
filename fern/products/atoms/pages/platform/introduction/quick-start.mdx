sidebarTitle: "Quick start"

---

Pick a template, fill in your details, and talk to your agent. No prompt writing needed to get started.

<Steps>
  <Step title="Open Create Agent">
    From your [dashboard](https://app.smallest.ai/), click the green **Create Agent** button in the top right.

    <Frame caption="Dashboard with Create Agent button">
      <img src="../building-agents/images/create-agent-modal.png" alt="Create Agent modal with creation options" />
    </Frame>
  </Step>

  <Step title="Choose Start with Template">
    Select **Start with Template**, then choose **Single Prompt** as your agent type.

    <Frame caption="Choose Start with Template from the Create Agent modal">
      <img src="../building-agents/images/choose-start-with-template.png" alt="Choose Start with Template from the Create Agent modal" width="700" style={{ borderRadius: '8px' }} />
    </Frame>
  </Step>

  <Step title="Pick a Template">
    Browse the template gallery and pick one close to your use case. Use **Filter By** to narrow by industry, direction (inbound or outbound), or agent type. Click a template, then hit **Create**.

    <Frame caption="Browse and select a template that fits your use case">
      <img src="../building-agents/images/browse-select-template.png" alt="Browse and select a template that fits your use case" width="700" style={{ borderRadius: '8px' }} />
    </Frame>
  </Step>

  <Step title="Configure the Basics">
    Before you touch the prompt, set these in the left panel:

    | Field | What to choose |
    |-------|----------------|
    | **Call Direction** | **Inbound** if customers call in, **Outbound** if the agent makes calls |
    | **Emotive Model** | Toggle on for a more expressive voice (Beta), or leave off |
    | **Voice** | Pick a voice from the library. Use the preview to listen first |
    | **Knowledge Base** | Optionally attach an existing KB so the agent can use your docs or FAQs |
  </Step>

  <Step title="Customize Your Template">
    The editor opens with a prompt, voice, and structure already filled in.

    <Frame caption="The editor opens with the template pre-configured">
      <img src="../building-agents/images/customize-template-editor.png" alt="The editor opens with the template pre-configured" width="700" style={{ borderRadius: '8px' }} />
    </Frame>

    Replace the placeholders with your own details:

    - **Company name and details.** Swap in your actual business name
    - **Policies and rules.** Update return windows, hours, pricing, and similar details
    - **Tone.** Adjust the wording to match your brand

    <Tip>
    Keep the section headers as they are. They keep both you and the AI organized, even after you change the content.
    </Tip>
  </Step>

  <Step title="Test Your Agent">
    Click **Test Agent** in the top right to start a test call. You can test three ways:

    - **Web Call.** Talk to your agent through your browser microphone
    - **Telephony Call.** Enter a phone number and get a call from your agent
    - **Chat.** Have a text conversation with your agent

    <Frame caption="Test your agent via Web Call, Telephony, or Chat">
      <img src="../../images/testing-voice-agent.png" alt="Test your agent via Web Call, Telephony, or Chat" width="700" style={{ borderRadius: '8px' }} />
    </Frame>

    Try a few scenarios: ask a normal question, ask something unexpected, and interrupt mid-response. Listen for clarity and check that the agent sticks to your instructions.
  </Step>
</Steps>
