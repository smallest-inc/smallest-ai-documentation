## Reporting Issues

When reaching out, include as much detail as possible so we can help faster.

<AccordionGroup>
  <Accordion title="Agent Issues">
    - Agent ID
    - Agent type (Single Prompt or Conversational Flow)
    - Description of expected vs. actual behavior
    - Steps to reproduce
    - Sample conversation transcript (if available)
  </Accordion>
  <Accordion title="Call Quality Issues">
    - Call ID
    - Phone number involved
    - Time of call
    - Description of the issue
    - Your location (for regional troubleshooting)
  </Accordion>
  <Accordion title="Platform Issues">
    - Browser and version
    - Steps to reproduce
    - Screenshots
    - Any error messages
  </Accordion>
</AccordionGroup>
