title: Drop-in mobile voice widgets for iOS, Android, React Native, and Flutter
---

Every mobile stack now has a drop-in voice widget. One component call renders a floating "Ask AI" pill on your host screen and opens a bottom-sheet voice session on tap. Same agent ID across platforms, same pipecat pipeline behind the scenes.

The widgets are thin clients over the public [Realtime Agent WebSocket](/atoms/api-reference/api-reference/realtime-agent/realtime-agent). No WebRTC dependency, no LiveKit SDK, no framework lock-in.
