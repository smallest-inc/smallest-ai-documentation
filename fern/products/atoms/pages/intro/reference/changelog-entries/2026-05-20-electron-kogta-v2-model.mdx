A new Electron Kogta v2 model is available on Atoms agents. It's gated to the same allowed-orgs list as Kogta v1 — no migration on existing v1 agents.

**How to use:**

- In the agent config, the LLM dropdown now includes **Electron Kogta v2** alongside v1 (visible only to allow-listed orgs).
- Existing agents on Kogta v1 continue working unchanged. Switch the dropdown to v2 to opt in for new turns.

**Migration:** no action. Opt in per agent by switching the model.
