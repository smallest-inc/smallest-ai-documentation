The file-upload tab on the **STT playground** in the Atoms console has been removed. The playground now focuses on the realtime / streaming transcription path.

**For batch / pre-recorded transcription**, use the API directly:

```bash
curl -X POST "https://api.smallest.ai/waves/v1/pulse/get_text?language=en" \
  -H "Authorization: Bearer $SMALLEST_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/audio.wav"}'
```

**Migration:** no action. If you were using the playground's upload tab to test transcribe-from-file, paste the cURL above into your terminal — same response shape.
