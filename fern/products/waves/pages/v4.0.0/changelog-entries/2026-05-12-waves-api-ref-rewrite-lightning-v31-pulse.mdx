The Lightning v3.1 (sync + SSE + WebSocket) and Pulse (REST + WebSocket) endpoints in the API reference have been rewritten end-to-end. Each page now opens with a one-paragraph **what**, a **when to use this** comparison against the alternatives, a how-it-works walkthrough, copy-paste examples for **cURL / Python / JavaScript**, and a **common gotchas** section drawn from real customer reports.

All five sets of examples were live-tested against `api.smallest.ai` before publishing.

A note for JavaScript users: the official `smallestai` npm package (v1.0.1) predates Lightning v3.1 and Pulse, so the docs show `fetch` (REST/SSE) and `ws` (WebSocket) examples for those endpoints — they work in Node and the browser with no SDK install.
