title: Fern CLI 5.28.2 + Python SDK generator 5.3.14
date: 2026-05-19
description: Fern CLI bumped to 5.28.2 and Python SDK generator bumped from 4.61.3 to 5.3.14.
---

The Fern CLI has been updated to **5.28.2** and the Python SDK generator to **5.3.14**.

The CLI upgrade ships a fix for `@fern-api/replay` that prevents customer commits on a fern-bot regen branch from being silently dropped when that branch is merged via a merge commit. The generator bump aligns future automated regens with the output already produced in the latest Python SDK pull request.
