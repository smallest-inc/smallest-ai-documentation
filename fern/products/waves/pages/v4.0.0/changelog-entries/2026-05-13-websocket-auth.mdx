---
title: WebSocket endpoints now document Bearer auth
---

The Lightning v2, Lightning v3.1, and Pulse STT WebSocket channels now explicitly declare `Authorization` as a connection parameter. Pass your API key as `Authorization: Bearer <token>` in the WebSocket handshake headers — the same credential used for REST requests.
